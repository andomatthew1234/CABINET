⚡ Installing THE CABINET

Get up and running in just a few minutes.

Step 1 — Install Python
THE CABINET requires Python to run games, launcher components and updates.
📖 Expand Instructions

Python is required before installing THE CABINET.

Open Python in Microsoft Store

After installing Python:

  1. Open Command Prompt.
  2. Type:
python --version

If a version number appears, Python has been installed correctly.

The Microsoft Store version automatically configures most settings, making installation easier.

Step 2 — Install Pygame Community Edition
Install the game engine used by THE CABINET and custom games.
📖 Expand Instructions

Open Command Prompt and run: 

pip install pygame-ce

If pip is unavailable: 

python -m pip install pygame-ce

Wait for installation to finish.

Verify pygame-ce is installed: 

pip show pygame-ce
Step 3 — Download and Run the Installer
Download install.bat from the CABINET GitHub repository.
📖 Expand Instructions

Open the CABINET GitHub repository:

Open install.bat on GitHub

GitHub does not always allow direct downloads from external websites, so use the GitHub page to download the file.

  1. Click the button above.
  2. GitHub will open the install.bat file.
  3. Click the Download raw file button near the top-right of the file viewer.
  4. Save install.bat to your computer.
  5. Locate the downloaded file.
  6. Double-click install.bat.
  7. Allow Windows to run the script if prompted.
  8. Wait for installation to complete.

The installer will automatically set up THE CABINET and its required folders.

Step 4 — Start Playing
Launch THE CABINET and enjoy offline retro gaming.
📖 Expand Instructions
  • Launch THE CABINET.
  • Browse installed games.
  • Play fully offline.
  • Use One-Click Update to stay up to date.
  • Import your own pygame-ce games whenever you want.

Depending on the version installed, games may include:

  • 🥊 StreetFight
  • 🏎️ NEONRACE
  • 🐍 Snake
  • 🧩 Tetris
  • 💧 Water Sort
  • 🏓 Pong
