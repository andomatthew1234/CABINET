⏬ Import Custom Games

Add your own pygame games into THE CABINET.

1. Create Your Game Folder

Drop your game folder into the /games/ directory. 

/
├── app.py
├── install.bat
├── LICENCES.MD
├── readme.md
├── games/
│   ├── your_game_folder_here/
│   │   ├── game.py
│   │   └── config.json
│   └── ...
├── music/
└── other/

2. Create game.py

This is your actual game.

THE CABINET expects a pygame-ce project. 

import pygame

pygame.init()

screen = pygame.display.set_mode((800,600))

running = True

while running:

    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    screen.fill((20,20,20))
    pygame.display.flip()

pygame.quit()

3. Create config.json

The launcher uses this file to display information about your game. 

{
    "title": "game_name",
    "description": "game_description",
    "high_score": 0
}

The high_score field is optional but recommended.

4. Launch THE CABINET

THE CABINET automatically scans the games folder and imports valid games into the launcher.

No coding changes to THE CABINET itself are required.
