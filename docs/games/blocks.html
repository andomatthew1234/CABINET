🧱 Block Blast

Strategic 8x8 Grid Puzzle

A modern single-player puzzle game combining the best elements of Tetris and Woodoku. Drag and drop block shapes onto the 8x8 grid to clear rows and columns. Earn massive combo points and survive as long as you can before running out of space!

  • 🧩 Drag-and-drop gameplay with smooth animations and grid snapping
  • 💥 Simultaneous row and column clearing with satisfying particle effects
  • 📈 Dynamic combo multiplier system for consecutive line clears
  • 🎨 Clean modern visual style with bright colors, soft shadows, and pop-up text
